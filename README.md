# FIVE BY SIX

A word guessing game. Create a game link with a five letter word and share it for others to guess.

https://fivebysix.co.uk

## Stack
- React + Tailwind frontend.
- AWS serverless static hosting (Optional).

## Run local
- NodeJS (v20 ro more).
- `npm install`
- `npm run dev`

## Build
- `npm run build`

## Test
- `npm run test`
